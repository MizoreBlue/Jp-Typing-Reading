# JP-Typing-Reader 双语轻小说打字阅读器 产品需求文档

## 1. 产品概述

JP-Typing-Reader 是一款专为日语学习者设计的 Web 应用程序，通过"双语对照阅读+沉浸式打字练习"模式，实现阅读能力提升、打字技能训练与词汇记忆巩固的三位一体学习体验。

### 核心价值
- **边读边打**：在阅读日语轻小说过程中，通过键盘输入罗马音转化为假名/汉字
- **双语对照**：日语原文与中文译文同步展示，段落级精准对齐
- **注音辅助**：所有汉字自动标注假名读音，降低阅读门槛
- **词库巩固**：自动收集错题与生词，生成个性化复习本

### 目标用户
- 初级~中级日语学习者（N5~N2 水平）
- 轻小说爱好者
- 希望提升日语打字速度的学习者

## 2. 核心功能

### 2.1 功能模块总览

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| 双书导入模块 | 上传日语原文 EPUB 和中文译文 EPUB | P0 |
| 段落对齐模块 | 自动/手动对齐中日段落 | P0 |
| 打字练习模块 | 罗马音转假名/汉字的沉浸式输入 | P0 |
| 双语阅读模块 | 同步滚动的双语对照界面 | P0 |
| 注音模块 | 汉字自动标注假名读音 | P0 |
| 生词本模块 | 错词收集与复习 | P1 |
| 进度管理 | 阅读位置记忆与恢复 | P1 |

### 2.2 核心页面

1. **首页/导入页**
   - 文件上传区域（日语 EPUB + 中文 EPUB）
   - 已导入书籍列表
   - 最近阅读记录

2. **对齐调整页**
   - 左右双栏段落对比
   - 拖拽式段落关联
   - 对齐预览与保存

3. **阅读练习页（主页面）**
   - 左侧：日语原文打字区（带 ruby 注音）
   - 右侧：中文译文对照区
   - 顶部：章节导航与进度条
   - 底部：打字统计（速度、正确率）

4. **生词本页**
   - 错词列表（按错误频率排序）
   - 单词详情卡片
   - 复习模式入口

## 3. 核心流程

### 3.1 用户主流程

```mermaid
flowchart TD
    A[导入日语 EPUB] --> B[导入中文 EPUB]
    B --> C[自动段落对齐]
    C --> D{对齐结果满意?}
    D -->|是| E[开始阅读练习]
    D -->|否| F[手动调整对齐]
    F --> E
    E --> G[打字阅读日语原文]
    G --> H[查看中文译文]
    H --> I[遇到生词/错词]
    I --> J[加入生词本]
    J --> G
    G --> K{章节完成?}
    K -->|否| G
    K -->|是| L[进入下一章]
```

### 3.2 打字输入流程

```mermaid
flowchart LR
    A[用户输入罗马音] --> B[实时转换为假名]
    B --> C{匹配汉字读音?}
    C -->|是| D[自动替换为汉字]
    C -->|否| E[保持假名显示]
    D --> F[显示正确反馈]
    E --> G[显示假名]
    G --> H{用户继续输入?}
    H -->|是| A
    H -->|否| I[等待下一个词]
```

## 4. 用户界面设计

### 4.1 设计风格

**设计理念：和风简约 · 沉浸阅读**

- **配色方案**：
  - 主色：深靛蓝 `#1a365d`（沉稳书卷气）
  - 辅色：米白 `#faf5eb`（纸张质感）
  - 强调色：樱粉 `#f0a5b8`（进度/高亮）
  - 文字色：墨黑 `#1a1a1a` / 灰 `#4a5568`

- **字体选择**：
  - 日文标题：Noto Serif JP（衬线体，优雅端正）
  - 日文正文：Noto Sans JP（无衬线，清晰易读）
  - 中文正文：思源宋体 / 系统默认宋体
  - 代码/罗马音：JetBrains Mono

- **布局风格**：
  - 双栏对照布局（左日右中）
  - 卡片式章节列表
  - 底部固定统计栏
  - 圆角边框 + 微妙阴影

- **动效设计**：
  - 打字正确：字符轻微弹跳 + 绿色闪烁
  - 打字错误：红色边框抖动
  - 页面切换：淡入淡出过渡
  - 滚动同步：平滑滚动动画

### 4.3 响应式策略

- **桌面优先**（1024px+）：双栏并排显示
- **平板适配**（768px-1023px）：双栏稍窄，保持并排
- **移动端暂不支持**（<768px）：显示提示"请使用电脑体验完整功能"

## 5. 数据管理

### 5.1 存储策略

| 数据类型 | 存储位置 | 说明 |
|---------|---------|------|
| 用户偏好设置 | LocalStorage | 主题、字体大小、注音开关 |
| 阅读进度 | LocalStorage | 当前章节、字符位置 |
| 对齐数据 | IndexedDB | 段落映射关系（大文本） |
| 生词本 | IndexedDB | 错词列表、错误次数 |
| 书籍元数据 | IndexedDB | 书名、作者、章节列表 |

### 5.2 数据导出/导入

- 支持将对齐数据导出为 JSON 文件
- 支持导入已有对齐数据
- 支持生词本导出为 CSV

## 6. 验收标准

### 6.1 功能验收

- [ ] 可成功解析标准 EPUB 文件（2.0/3.0）
- [ ] 段落对齐准确率 ≥ 85%（标准排版轻小说）
- [ ] 打字延迟 ≤ 100ms
- [ ] 注音显示正确（ruby 标签）
- [ ] 进度保存与恢复正常
- [ ] 生词本收集与显示正常

### 6.2 浏览器兼容性

- Chrome 90+
- Firefox 88+
- Edge 90+
- Safari 14+（可选）

### 6.3 性能指标

- 首屏加载时间 ≤ 3s
- 打字响应时间 ≤ 100ms
- 内存占用 ≤ 500MB（处理 10 万字文本时）
